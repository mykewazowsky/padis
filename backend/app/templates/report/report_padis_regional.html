Ringkasan Analisis
Wilayah {{ region }} pada skenario {{ scenario_label }} dengan kondisi {{ climate_label }} mencatat estimasi total kerugian sebesar {{ total_loss }}. {% if aal_pct_up %} Perubahan iklim meningkatkan risiko tahunan rata-rata (AAL) sebesar {{ aal_pct }} dari {{ aal_nonclimate }} menjadi {{ aal_climate }}, dengan selisih absolut {{ aal_delta }}. {% else %} AAL pada skenario Climate ({{ aal_climate }}) lebih rendah {{ aal_pct }} dibanding baseline Non-Climate ({{ aal_nonclimate }}), selisih absolut {{ aal_delta }}. {% endif %}
Implikasi Praktis
  • Nilai kerugian (Loss) mencerminkan estimasi risiko untuk satu skenario return period ({{ scenario_label }}). Gunakan nilai AAL untuk mengukur beban risiko tahunan yang lebih representatif.
  • {% if aal_pct_up %}Peningkatan AAL sebesar {{ aal_pct }} mengindikasikan bahwa perubahan iklim meningkatkan eksposur risiko pertanian padi di wilayah ini. Intervensi adaptasi iklim perlu dipertimbangkan. {% else %}Penurunan AAL sebesar {{ aal_pct }} menunjukkan risiko relatif lebih rendah pada skenario iklim. Tetap lakukan pemantauan berkala karena dinamika risiko dapat berubah seiring waktu.{% endif %}
  • Hazard Index dapat digunakan sebagai indikator prioritas kawasan untuk kajian lanjutan atau intervensi mitigasi berbasis kerawanan lokal.
  • Laporan regional ini bersifat profil wilayah tunggal, bukan perbandingan antarwilayah. Untuk perbandingan nasional, gunakan laporan tingkat nasional.
{% endif %}
Definisi & Catatan
Loss — Estimasi kerugian ekonomi akibat kejadian bencana untuk return period {{ scenario_label }} (Rp).
AAL — Average Annual Loss, rata-rata kerugian tahunan yang diharapkan (Rp/tahun).
Hazard Index — Indeks bahaya ternormalisasi 0–1. Nilai mendekati 1 = bahaya lebih tinggi.
Total Produksi — Total produksi padi historis kabupaten/kota (ton).

Laporan ini dihasilkan otomatis oleh PADIS berdasarkan data model terbaru. Ditujukan sebagai alat bantu analisis, bukan dokumen resmi kebijakan.

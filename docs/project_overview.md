# PADIS - Project Overview

## 1. Judul Proyek
PADIS: Paddy Loss Information System

## 2. Tujuan
Membangun WebGIS interaktif untuk menganalisis kerugian produksi padi dan kerugian ekonomi akibat banjir, kekeringan, dan multi-hazard pada tingkat kabupaten/kota di Indonesia.

## 3. Input Data
- Batas administrasi kabupaten/kota
- Tutupan lahan sawah
- Raster bahaya banjir
- Raster kekeringan (SPI)
- Data produksi padi
- Harga gabah kering

## 4. Proses Utama
- Analisis hazard banjir
- Analisis hazard kekeringan
- Analisis multi-hazard
- Estimasi kehilangan produksi
- Estimasi kerugian ekonomi

## 5. Output Dashboard
- Peta interaktif
- Ringkasan statistik
- Tabel hasil analisis
- Grafik
- Detail per wilayah

## 6. Unit Analisis
Kabupaten/Kota

## 7. Output Akhir
- Indeks bahaya banjir
- Indeks bahaya kekeringan
- Indeks multi-hazard
- Estimasi kehilangan produksi padi
- Estimasi kerugian ekonomi dalam rupiah

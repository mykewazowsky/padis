Definisi Variabel
Loss — Estimasi kerugian ekonomi akibat kejadian bencana untuk satu return period tertentu (Rp).
AAL (Average Annual Loss) — Rata-rata kerugian tahunan yang diharapkan dengan memperhitungkan distribusi probabilistik return period (Rp/tahun).
Hazard Index — Indeks bahaya ternormalisasi (0–1), semakin mendekati 1 semakin tinggi tingkat bahaya.
Return Period — Periode ulang kejadian bencana. Contoh: RP25 = kejadian dengan probabilitas terlampaui 4% per tahun.
Catatan Penggunaan Laporan
Laporan ini dihasilkan secara otomatis oleh sistem PADIS berdasarkan data model terbaru. Nilai Loss dan AAL merupakan estimasi berdasarkan model risiko probabilistik pertanian padi di Indonesia. Laporan ini ditujukan sebagai alat bantu analisis, bukan sebagai dokumen resmi kebijakan. Untuk pertanyaan metodologi, hubungi tim teknis PADIS.
